HazeNet ⚔️ AFNO + StormCast

วิเคราะห์เทียบจุดต่อจุด แบบตรงไปตรงมา — งานเราดีพอหรือยัง และจะขึ้น SOTA ได้ยังไง

เอกสารวิเคราะห์ต่อเนื่องจาก HazeNet_Pipeline_Explained.html · ถอดบทเรียนจากโปสเตอร์ AFNO + StormCast (พยากรณ์อากาศไทย ทดสอบกับไต้ฝุ่นนอรู) มาเทียบกับโปรเจกต์เรา

เรานำด้าน: ความใหม่ + ฟิสิกส์ เขานำด้าน: ปิดจบ + สื่อสาร เป้า: เอาจุดแข็งทั้งสองมารวมกัน

ข้อสรุปใน 30 วินาที: งาน AFNO+StormCast ไม่ใช่ SOTA ของโลก — มันคือ "งานต่อเครื่องมือ NVIDIA 2 ตัวแล้วเอามาจับไทย" ที่ปิดจบสวยและขายเป็น · งานเราทะเยอทะยานและใหม่กว่าเชิงวิทยาศาสตร์มาก (มี attribution + counterfactual + physics guarantee ที่เขาไม่มีเลย) · แต่เราตามหลังเขาเรื่อง "ผลลัพธ์ชนะที่สะอาด + การสื่อสารที่คม + การปิดจบให้ทันเวลา" — บทเรียนสำคัญสุดคือ diffusion (StormCast) คือยาแก้จุดอ่อนที่ใหญ่ที่สุดของเรา (จับพีควันวิกฤต)

4. จุดอ่อนงานเรา (พูดตรง ไม่กลบ)

  1. ยังไม่มี "ตารางชนะที่สะอาด" สักใบ จุดอ่อนเบอร์ 1
    ทั้งโปสเตอร์เขายืนบนตาราง ACC ใบเดียว เรามี MAE 55 ที่ "ยังไม่ชนะใครแบบเห็นภาพ" และที่แย่กว่า — ยังไม่ได้รัน PM2.5-GNN/AirPhyNet จริง ถ้าวันส่งยังเทียบแค่ XGBoost/mean เราจะโดนตกแบบเดียวกับที่โปสเตอร์เขาเสี่ยง (เทียบแค่ตัวเอง)
  2. Inversion พัง (r = −0.038) ต้องตัด
    ผลลบบนฟีเจอร์ที่เคยโฆษณา → ต้องตัดทิ้งให้สะอาดหรือลดเป็น proof-of-concept เคสไฟจุดเดียว อย่าปล่อยเลขติดลบในรายงาน
  3. Attribution ยัง "ดูสมเหตุสมผล" เฉยๆ ยังไม่ validate เสี่ยงโดนแทง
    ถ้าไม่เทียบ WRF-Chem/PMF จริง มันคือ "รูปสวยที่พิสูจน์ไม่ได้" = จุดที่กรรมการแทงง่ายสุด
  4. จุดพีค/วันวิกฤต = คุณค่าหลักเรา แต่เป็นส่วนที่ AI ทำยากที่สุด กำแพงใหญ่
    ดูโปสเตอร์: ตัวแปร sparse + heavy-tail (ฝน) เขาได้ ACC แค่ 0.115 = แทบใช้ไม่ได้ — PM2.5 พีควันวิกฤต คือ "ฝน" ของเรา เรากำลังโจมตีส่วนยากที่สุดเป็นแกนหลัก ถ้าไม่มีกลยุทธ์เฉพาะ จะชนกำแพงเดียวกับที่เขายอมแพ้
  5. สโคปใหญ่เกินจะปิดจบ เสี่ยงไม่ทัน
    เขาทำ 1 อย่างให้เสร็จ เรามี 5–6 อย่าง → ความเสี่ยง "ทำครบแต่ตื้นทุกอัน" สูงมาก

5. เรียนรู้อะไร + เอามาปรับใช้ทันที

🔑 บทเรียน 1 — สร้าง "Money Figure" 2 ใบ แล้วถอยหลังมาทำงาน

ทั้งโปสเตอร์เขายืนบน (ก) ตาราง ablation ชนะทุกตัว (ข) case study ไต้ฝุ่น · เรากำหนด money figure ของเราเดี๋ยวนี้:

📊 ใบที่ 1 — ตาราง ablation ชนะทุกช่อง

แถว: CLNO เปล่า → +emission curve → +BLH → เทียบ GNN/AirPhyNet
คอลัมน์: MAE · RMSE · bias ปีต่ำ · CSI วันวิกฤต
เป้า: ลด bias ปีต่ำ +45→≤25 + ชนะทุก baseline

🗺️ ใบที่ 2 — case study วันเชียงใหม่อันดับ 1 โลก

วันที่ มี.ค. 2023 ที่ CM ติดอันดับ 1 อากาศแย่สุดโลก → แผนที่พยากรณ์ + แผนที่ attribution "ควัน X% มาจากเมียนมา" = ไต้ฝุ่นนอรูเวอร์ชันเรา

🔑 บทเรียน 2 — Diffusion คือ "ยา" แก้จุดอ่อนใหญ่สุดของเรา สำคัญสุด

เราเคยตัดสินใจไม่ใช้ diffusion เพราะมันทำลาย superposition + ช้า — ซึ่งถูกต้องสำหรับแกน attribution แต่โปสเตอร์นี้พิสูจน์ว่า diffusion คือวิธี SOTA ที่แก้อาการ "AI ทำนายเบลอ/พลาดจุดพีค" ได้จริง (ดู rh_2m พุ่ง 0.55→0.70 เพราะ StormCast)

AFNO เปล่า = วาดภาพด้วยพู่กันใหญ่ → ได้ภาพเบลอๆ ค่าเฉลี่ย · diffusion = เอาพู่กันเล็กมาเก็บรายละเอียดคมๆ กู้ "ยอดแหลม" กลับมา · นี่คือสาเหตุที่ฝน(ค่าพีคแหลมๆ) ดีขึ้นเมื่อเติม StormCast
อาการที่เราเคยกลัว (ทำนายเบลอ underpredict พีค จากการใส่ noise) → diffusion คือยาแก้โดยตรง เราตัดมันทิ้งเร็วเกินไป ควรนำกลับมาแบบมีเงื่อนไข (ดูข้อ 6)

🔑 บทเรียน 3 — กล่อง Advection ของเขายืนยันว่าเรามาถูกทาง

"งานที่สำเร็จ" ก็ใช้ operator splitting + physics advection เหมือนเรา → อย่าไปสงสัยแกนตัวเอง จุดต่างของเราคือทำให้มันเชิงเส้นเพื่อ attribution ซึ่ง defend ได้

🔑 บทเรียน 4 — สเปก + ความละเอียดที่ขายได้

เขาขาย "3×3 กม." เราอยู่ 11 กม. → ถ้าทำหัว diffusion (ข้อ 6) จะได้ downscaling มาฟรี → พูดได้ว่า "พยากรณ์ + ระบุแหล่ง ระดับ ~3 กม. ใน X วินาที"

🔑 บทเรียน 5 — บีบสโคปให้เหลือ "1 unit ที่ปิดจบสวย"

แกนต้องเสร็จ 100%: พยากรณ์ชนะ baseline + attribution validated + 1 case study + error budget · ส่วน inversion/GFS/twin = roadmap slide

6. 🚀 ไอเดียก้าวข้ามทั้งสองงาน — สถาปัตยกรรม "2 หัว"

นี่คือจุดที่ทำให้เราไม่ใช่แค่ลอกเขา แต่ก้าวข้ามทั้งเขาและตัวเราเดิม — เก็บแกนเชิงเส้นไว้ทำ attribution + เพิ่มหัว diffusion มากู้จุดพีค โดยแยกหน้าที่ชัดเจนไม่ให้พังกัน

กฎเหล็กของการต่อ 2 หัว (กันพัง)

ประเด็นวิธีกันไม่ให้พัง
diffusion ทำลาย superposition?หัว 2 ทำงานกับ ค่าพยากรณ์รวม (total) เท่านั้น — ไม่แตะ tracer แยกสี → attribution ยังมาจากหัว 1 ที่เชิงเส้น 100%
diffusion ช้า ทำลายเคลม "วินาที"?รัน diffusion เฉพาะ output สุดท้าย ไม่ใช่ทุก rollout step + ใช้ few-step sampler → ยังเร็ว
พิสูจน์ว่ามันช่วยจริง?รายงานเป็น ablation: CLNO เปล่า vs CLNO+diffusion บน CSI วันวิกฤต — เลียนแบบตาราง AFNO vs AFNO+StormCast ของเขาเป๊ะ
ถ้าทำไม่ทัน?หัว 2 เป็นของเสริม ตัดได้ — แกน attribution ไม่ตายตาม (ตรงตามกฎ "แกนห้ามตายเพื่อของเสริม")
ทำไมนี่คือ SOTA จริง: AFNO+StormCast มี diffusion แต่ไม่มี attribution · HazeNet เดิมมี attribution แต่ไม่มี diffusion (จับพีคไม่ได้) · HazeNet 2 หัว = ตัวแรกในโลกที่มีทั้งคู่ สำหรับควันข้ามแดน

7. ทำยังไงให้ขึ้น SOTA จริง (ไม่ใช่แค่ชนะโปสเตอร์นี้)

โปสเตอร์นี้คือ benchmark ด้าน "execution + communication" ไม่ใช่ด้าน novelty — เราชนะด้าน novelty อยู่แล้ว สิ่งที่ทำให้เราเป็นงานวิจัยระดับโลกจริง คือเอา 3 ของที่เรามี/เขาไม่มี มาทำให้ "พิสูจน์ได้สะอาดเท่าตาราง ACC ของเขา":

  1. ✅ Exact attribution ที่ validate เทียบ WRF-Chem + PMF → ไม่มี AI มลพิษตัวไหนในโลกทำ
  2. ✅ Counterfactual ที่ validate เทียบ source-off → ของจริงที่ใช้ตัดสินนโยบาย
  3. ✅ Error budget แบบ oracle substitution → บอกได้ว่า error มาจากไหน (เขาบอกไม่ได้เลย)
  4. 🚀 + จับพีควันวิกฤตด้วยหัว diffusion → จุดที่ฝน ACC 0.115 ของเขาล้มเหลว ถ้าเราจับ "วันเชียงใหม่อันดับ 1 โลก" ได้ = ความเหนือกว่าที่จับต้องได้

8. ✅ Action ที่ควรทำตอนนี้ (เรียงตามความสำคัญ)

  1. รัน PM2.5-GNN + AirPhyNet จริง ให้ได้ตารางเทียบภายนอก — ห้ามเทียบแค่ตัวเอง (กับดักของโปสเตอร์นี้)
  2. ปิด gate emission curve + BLH ให้ bias ปีต่ำ +45→≤25 → ออกมาเป็น ตาราง ablation ชนะทุกช่อง (money figure #1)
  3. ทำ case study วันเชียงใหม่อันดับ 1 โลก + แผนที่ attribution (money figure #2)
  4. ตัดสินใจหัว diffusion ที่ 2 — เก็บ CLNO ทำ attribution + เพิ่ม diffusion refinement กู้จุดพีค (ไอเดียที่ทำให้เหนือทั้งสองงาน)
  5. ตัด inversion ที่พัง ออกจากแกน ลดเป็น PoC/roadmap
  6. ทำโปสเตอร์ 4 ช่องแบบเขา (Objective/Method/Results/Conclusion) — เราเก่งเรื่องลึกอยู่แล้ว ขาดแค่ "ปิดจบให้คม"
สรุปบรรทัดสุดท้าย: งาน AFNO+StormCast สอนเราว่า "ความสำเร็จ = เสร็จ + ชนะสะอาด + สื่อสารคม" ไม่ใช่ "ทะเยอทะยานสุด" · เราใหม่กว่าเขาอยู่แล้ว สิ่งที่ขาดคือวินัยการปิดจบของเขา · เอา 3 จุดแข็งเรา (attribution + counterfactual + error budget) มาพิสูจน์ให้สะอาด + ยืม diffusion มากู้จุดพีค = เราจะไม่ใช่แค่ชนะโปสเตอร์นี้ แต่เป็น SOTA ใหม่ของโลกสำหรับควันข้ามแดน
