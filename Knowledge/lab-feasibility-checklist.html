4. กับดัก + caveat ของเฟสนี้

อย่าหลงทาง:
  • เทส ≠ ทำจริง — ทุกอย่างชั้น 0–4 ใช้ข้อมูลจิ๋ว/เครื่องฟรี เป้าคือ "รันได้ไหม" ไม่ใช่ "แม่นไหม" · อย่าตัดสินวิทยาศาสตร์จากเฟสนี้
  • อย่าข้ามชั้น 1 — น่าเบื่อแต่เป็นตัวฆ่าโปรเจกต์เงียบที่สุด (เลือกสัปดาห์ที่สถานีดับ = เสียเวลาทั้งสาย)
  • ERA5/CDS เป็นคอขวดเวลา — เริ่มสมัคร + ยิง request วันแรกเลย เพราะคิวรอนาน ไม่ใช่เรื่องโค้ดแต่เป็นเรื่องเวลา
  • อย่าทำชั้นใดให้สมบูรณ์แบบก่อนผ่านประตู — "พอใช้ได้" แล้วไปประตูถัดไป · เก็บการจูนละเอียดไว้ตอน run จริง
  • อย่าแตะ A100 ก่อนประตู 3 — ทั้ง 3 ประตูทำบนของฟรีได้ → เก็บงบ 15 ชม.ไว้ run จริงตอนทุกอย่างนิ่ง (โยง 4.5)
